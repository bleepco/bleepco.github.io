[info for nerds] — the setup’s cheap lo-tek: a modest box, some AI tools duct-taped together, and a bunch of improvised processes to keep it alive. If you code, it’ll probably make your eyes roll, cos this really isn’t about clever engineering (... or is it?)

  • Raspberry Pi 3B+ — 1.4GHz quad-core ARM Cortex-A53, 1GB RAM.
  • Custom batch renaming + metadata embedding tool to keep the library neat.
  • Lightweight, low-load visualizer built for continuous streaming.
  • AI-driven content generator and scripting pipeline glued together with simple shell scripts.
  • Stream served via ffmpeg loop — nothing fancy, just sturdy.
